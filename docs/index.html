OpenAI Codex

Codex 是 OpenAI 打造的 AI 編碼代理，具備自然語言理解、程式碼生成與自動化開發流程協助能力，幫助團隊從構思到交付保持高速與高品質。

前往 GitHub 探索完整指南

核心價值

自然語言到程式碼

透過 Codex，開發者可以使用自然語言描述需求並生成精準程式碼，支援 Python、JavaScript 等主流語言，大幅縮短構思到實作的距離。

全專案級智能

新版 Codex 能檢視、編輯與重構整個專案，支援測試、自動修復與差異摘要，為團隊帶來端到端的 AI 開發夥伴。

多場景整合

同時提供 CLI、IDE Plugin 與雲端 Codex Web，結合沙盒執行、MCP 擴充與自訂工作流程，適用個人開發到企業合作。

快速開始

npm install -g @openai/codex
codex

或使用 Homebrew：brew install codex。啟動後選擇以 ChatGPT 帳戶登入，即可串連現有的 Plus、Pro、Team、Edu 或 Enterprise 訂閱方案。

01
安裝 CLI

全球套件管理支援 npm、Homebrew 與 GitHub Release 二進位檔，適配 macOS、Linux 及 Windows。

02
登入 Codex

選擇以 ChatGPT 方案登入或使用 OpenAI API 金鑰；支援無頭環境與零資料保留 (ZDR)。

03
啟動專案助理

Codex 透過 CLI 或 IDE 擴充載入專案，支援多代理記憶、測試腳本與自動 PR 摘要輸出。

支援的工作流程

Codex CLI 可在本機安全沙盒中執行，並可透過 Model Context Protocol (MCP) 連結外部服務：

  • 撰寫或重構程式碼、補齊測試與文件
  • 執行 `codex exec` 進行非互動式自動化流程
  • 整合 GitHub Action、TypeScript SDK，打造自動化管線
  • 透過 `AGENTS.md` 建置持久記憶與專案守則

能力地圖

智慧合作

理解專案脈絡、維護者規範與測試策略，協助產出高品質的程式碼片段與評審建議。

自動化交付

支援自動執行測試、生成變更摘要與 PR 描述，縮短從需求提交到發布的週期。

安全沙盒

採用多層沙盒與權限審核，確保關鍵工作流程（如封存、部署）具備可控性與稽核紀錄。

Codex CLI 面世

提供跨平台的本機命令列體驗，支援 npm、Homebrew 與直接下載安裝。

IDE 整合

推出 VS Code、Cursor、Windsurf 擴充，將 Codex 功能帶入開發者的日常工作流程。

Codex Web

在 ChatGPT 進階方案中提供雲端版代理，支援跨專案協作與遠端執行環境。
